# 📊 Workforce Analytics Dashboard

## 📌 Project Overview
This project is a **Workforce Analytics Dashboard** built using **SQL, Python, and Tableau**. It provides insights into employee data across HR, performance, projects, and time & attendance to support data-driven decision-making.

The dashboard is designed with a **UI-style navigation system in Tableau**, making it feel like an interactive application rather than a static report.

---

## 🎯 Key Objectives
- Analyze workforce distribution across departments  
- Track employee performance and attrition risk  
- Monitor project status, health, and industry trends  
- Evaluate productivity and attendance patterns  
- Provide actionable business insights  

---

## 🛠 Tools Used
- SQL – Data extraction and querying  
- Python – Data preprocessing and preparation  
- Tableau – Data visualization and interactive dashboards  

---

## 📊 Dashboards Included
1. Workforce Overview  
2. Performance & Attrition Analytics  
3. Project Performance & Delivery Insights  
4. Time & Attendance Analytics  
5. Home Navigation Dashboard (UI-style)

---

## 🎨 UI / Design Feature
- Built a **UI-like navigation system inside Tableau**
- Each dashboard is connected through **interactive buttons**
- Provides a **seamless app-like experience for users**
- Designed for easy navigation between analytics views

---

## 📈 Key Insights Generated
- Workforce distribution across departments and roles  
- Employee performance segmentation (Excellent, Good, Average, Poor)  
- Attrition risk identification (Low, Medium, High)  
- Project health and budget utilization analysis  
- Productivity and overtime trends  

---

## 🚀 Outcome
- Improved understanding of workforce behavior  
- Identified performance and retention gaps  
- Analyzed project delivery efficiency  
- Enabled business-friendly storytelling through dashboards
